## 🇬🇧 English Version

### 🚀 Overview
**FLASH AI** is a high-level, self-evolving autonomous agent system built entirely in the **Ring language**. It functions as a **Senior Systems Architect**, capable of formal deterministic execution, self-healing code reflection, and modular expansion through dynamic tool creation.

Version 3.0 represents a massive architectural leap, introducing a **Formal Finite State Machine (FSM)** for execution stability, **Context Intelligence** for semantic memory weighting, and a **Reflection Engine** for auto-healing from API and code errors.

### ✨ Key Features
- **⚙️ Deterministic FSM Architecture:** Replaced legacy loops with strict state machine verification (`AGENT_PLANNING`, `AGENT_EXEC_TOOL`, `AGENT_REFLECTING`).
- **🧠 Context & Intelligence Layer:** Semantic scoring engine weights user errors and intents, guaranteeing deterministic context limits.
- **🔄 Reflection Auto-Correction:** Failsafe auto-reflection loop analyzes execution errors and instructs the agent how to fix them dynamically.
- **📊 Cumulative Token Tracking:** Accurately sums tokens across all internal thought steps.
- **🎨 Premium GUI Architecture:** Modern dark-mode interface with chat bubbles and reasoning blocks.
- **🛠️ Domain-Modular Arsenal:** Tools properly isolated into file, code, project, and system sub-modules.

### 🏗️ Getting Started
#### Prerequisites
- **Ring Language 1.18+** (with `guilib`, `curllib`, `jsonlib`).
- **API Keys:** Place keys for Gemini/OpenAI in `config/api_keys.json`.

#### Execution
To run the **Graphical Interface (Recommended)**:
```bash
ring gui_main.ring
```
To run the **Terminal Interface (CLI)**:
```bash
ring main.ring
```

##  النسخة العربية

### 🚀 نبذة عن المشروع
**FLASH AI** هو نظام وكيل ذكي مستقل عالي المستوى، مبني بالكامل بلغة **Ring**. يعمل النظام كمهندس نظم أقدم (Senior Systems Architect)، قادر على اتخاذ القرارات بشكل صريح (Deterministic FSM)، المعالجة الذاتية للأخطاء، والتوسع من خلال إنشاء أدوات برمجية بشكل ديناميكي.

تمثل النسخة **3.0** قفزة معمارية هائلة، حيث تقدم **آلة حالات محدودة (FSM) لضمان استقرار التنفيذ**، إضافة لـ **طبقة ذكاء للذاكرة (Context Intelligence)** تقوم بوزن الاهتمام، ومحرك **التفكير العكسي (Reflection Engine)** للمعالجة الذاتية للأخطاء.

### ✨ المميزات الرئيسية
- **⚙️ بنية آلة الحالات (FSM):** استبدال أنظمة الدوران القديمة بنظام تنقّل آمن ومنظّم (التخطيط، التنفيذ، المراجعة).
- **🧠 طبقة الذكاء والسياق:** محرك تقييم دلالي يعطي أولوية للمشاكل وأسئلة المستخدمين بدلاً من الاعتماد على الطول فقط.
- **🔄 التصحيح الذاتي:** في حال فشل أي أداة، يُحلّل النظام رسالة الخطأ تلقائياً ويوجه الذكاء الاصطناعي لحله دون تدخل بشري.
- **📊 حساب تراكمي للكلمات:** حساب دقيق لتكلفة كل جلسة شامل العمليات المتعددة بالخلفية.
- **🎨 واجهة رسومية فاخرة:** تصميم عصري مظلم ومريح للمطورين.
- **🛠️ أدوات مقسّمة معماريّاً:** تقسيم جميع الأدوات بطريقة نموذجية معزولة للحصول على أفضل أداء برمجي.

### 🏗️ كيف تبدأ؟

- لغة **Ring 1.18** أو أحدث (مع مكتبات `guilib`, `curllib`, `jsonlib`).
- مفاتيح الـ API في ملف `config/api_keys.json`.

#### التشغيل
لتشغيل **الواجهة الرسومية (الخيار الأفضل)**:
```bash
ring gui_main.ring
```
لتشغيل **واجهة سطر الأوامر (CLI)**:
```bash
ring main.ring
```
